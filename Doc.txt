Program Description:

This program allows the user to simulate a ticketing system(pre-sales) for a cinema theater. The program begins with asking the user to purchase between 1-4 tickets per transaction. After each purchase, the program displays the purchased amount of tickets and those remaining for availability. The program continues until all tickets are sold. The program ends with displaying the user the total number of buyers involved in ticket transactions.

Function: get_ticket_request

Displays the tickets remaining and prompts the user to enter the amount of tickets they want to purchase.

Function: sell_ tickets

Controls the ticket-selling process by repeatedly collecting the users inputs, validating request, updating the remaining tickets and counting the number of buyers involved until all tickets are sold.

	1.	Initialize TOTAL_TICKETS to 20.
